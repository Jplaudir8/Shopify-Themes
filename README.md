# Shopify-Themes
It is important to note that shopify themes follow the same structure in terms of directories. This conventional tree structure is as follows:

```
[project-name]
│ 
└─── assets
│       application.js
│       application.scss.liquid
│   
└─── config
|       settings_data.json
|       settings_schema.json
|   
└─── layout
|       theme.liquid
|
└─── locales
|       en.default.json
|
└─── templates
|   |
|   └─── customers
|           account.liquid
|           activate_account.liquid
|           addresses.liquid
|           login.liquid
|           order.liquid
|           register.liquid
|           reset_password.liquid
|   
|        404.liquid
|        article.liquid
|        blog.liquid
|        cart.liquid
|        collection.liquid
|        collection.list.liquid
|        gift_card.liquid
|        index.liquid
|        list-collections.liquid
|        page.contact.liquid
|        page.liquid
|        product.liquid
|        search.liquid
| config.yml


```
